# Beard Blueprint Structure Notes (for future website builds)

Source reviewed: `Grandparent_Stories_Business_Blueprint_beard.com...pdf`

## Core structure pattern to replicate

1. **Opportunity framing first**
   - Explain why now
   - Tie to emotional + market demand

2. **Mission statement**
   - One clear human outcome sentence

3. **Simple business model flow**
   - 4-6 concrete operational steps

4. **Offer ladder / packages**
   - Entry, Mid, Premium
   - Side-by-side pricing + inclusions

5. **Lean launch plan**
   - Sell first, spend later
   - Validate before tooling overhead

6. **Website blueprint sections**
   - Hero + CTA
   - How it works
   - Packages
   - Why/vision
   - Trust/about
   - Contact/booking form

7. **Marketing plan**
   - Specific channel + creative hook + budget
   - ICP targeting details

8. **Sales conversion process**
   - Follow-up speed
   - Call structure/script
   - Explicit close ask

9. **Production SOP**
   - Delivery workflow
   - Tool stack
   - Quality checks

10. **Automation + scale**
   - Lead magnet
   - Messaging automation
   - Reporting loop

## Adaptation rule for NGU / real estate projects

For future NGU pages, map the same logic to:
- Opportunity -> suburb demand + timing
- Mission -> best-fit buyer outcomes
- Packages -> advisory tiers (shortlist/inspection/offer support)
- How it works -> shortlist -> inspect -> negotiate -> settle
- Sales process -> enquiry SLA + inspection booking cadence
- SOP -> listing verification, price checks, media compliance, handover checklist
