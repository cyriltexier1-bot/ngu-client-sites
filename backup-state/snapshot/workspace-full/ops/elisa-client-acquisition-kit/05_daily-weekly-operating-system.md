# Daily + Weekly Operating System (Client Acquisition)

## Daily (Mon-Fri)

### Block 1 — Speed-to-lead (20 min, start of day)
- Clear all new inbound leads
- First response sent to 100% of new leads
- Goal: response within 15 min during active hours

### Block 2 — Follow-up power hour (60 min)
- Work overdue follow-ups first
- Execute sequence touches for no-response leads
- Log outcome + next action date for every touch

### Block 3 — Pipeline progression (30 min)
- Move qualified leads to next stage
- Flag blocked deals with reason
- Request missing docs/info from prospects

### Block 4 — Reactivation (20 min)
- Contact 5 aged leads/day
- Use reactivation script + new CTA

## Weekly (Friday, 45 min)

1. Review KPIs:
- New leads count
- Median response time
- Contact rate
- Booking rate
- Close rate
- Reactivation conversion

2. Identify top bottleneck:
- Slow first response
- Low contact success
- Weak qualification
- Poor follow-up consistency

3. Pick one improvement for next week:
- One process change only (avoid overloading)

4. Update templates/scripts:
- Keep best-performing messaging
- Retire weak variants

## Non-negotiables

- Every lead has: owner, status, next action date
- No lead sits idle > 48h without action
- Weekly review happens even during busy weeks
