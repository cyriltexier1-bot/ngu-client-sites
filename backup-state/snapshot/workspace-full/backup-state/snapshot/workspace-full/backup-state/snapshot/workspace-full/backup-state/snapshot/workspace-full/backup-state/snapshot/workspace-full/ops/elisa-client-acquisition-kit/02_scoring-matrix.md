# Scoring Matrix (Fast Diagnostic)

Score each **Yes = 1**, **No = 0** across 15 questions.

## Total score

- **0-5 (Critical):** Foundation missing; fix process hygiene first
- **6-10 (Developing):** Some structure, weak consistency
- **11-13 (Strong):** Mostly good, optimize throughput/conversion
- **14-15 (High-performing):** Move to scale/automation

## Priority dimensions

### A) Lead Capture & Hygiene (Q1, Q2, Q6)
- If <=1 Yes: Immediate CRM cleanup + mandatory fields + source tagging

### B) Speed-to-Lead (Q3, Q4, Q9)
- If <=1 Yes: Build rapid response templates + booking CTA + SLA

### C) Follow-up Engine (Q5, Q11, Q13)
- If <=1 Yes: Create timed sequence + daily follow-up block + reactivation list

### D) Qualification & Conversion (Q8, Q10, Q14)
- If <=1 Yes: Qualification rubric + stage conversion tracking + proof assets

### E) Management Rhythm (Q7, Q12, Q15)
- If <=1 Yes: Weekly review cadence + KPI dashboard + process SOP

## Red flags (immediate attention)

- Q3 = No (slow response)
- Q5 = No (no systematic follow-up)
- Q1 = No (no single source of truth)
- Q10 = No (no conversion visibility)

If any red flag appears, put first-week effort there before optimization work.
