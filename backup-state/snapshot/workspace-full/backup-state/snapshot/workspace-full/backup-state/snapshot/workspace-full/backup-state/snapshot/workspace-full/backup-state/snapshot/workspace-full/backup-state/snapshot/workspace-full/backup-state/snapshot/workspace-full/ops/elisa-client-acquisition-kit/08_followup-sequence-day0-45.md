# Follow-up Sequence (Day 0 → Day 45)

- Day 0: first response within 15 min (SMS + email)
- Day 1: short check-in + booking link
- Day 3: value touch (local market insight)
- Day 5: objection support (“happy to answer questions”)
- Day 7: social proof (case study/testimonial)
- Day 10: urgency nudge (availability/timing)
- Day 14: soft close note (“close file unless still interested”)
- Day 21: re-open with new angle (new listing/new buyer activity)
- Day 30: monthly check-in + CTA
- Day 45: final reactivation before long-term nurture

## Rules
- Every touch logs: date, channel, outcome, next step
- If no response after Day 14, reduce cadence
- If response received, move to active pipeline immediately
