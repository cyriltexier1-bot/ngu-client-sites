# Lead Qualification Scorecard (0-100)

Score each lead:

- Intent clarity (0-20)
- Timeline urgency (0-20)
- Budget fit / finance readiness (0-20)
- Decision readiness (0-20)
- Property-fit / business fit (0-20)

## Score bands
- 80-100: Hot → immediate priority, same-day call
- 60-79: Warm → nurture + booking push
- 40-59: Cool → value sequence + requalification
- <40: Long-term → low-touch nurture

## Mandatory data capture
- Full name
- Mobile/email
- Source
- Goal (buy/sell/rent/invest)
- Target timeframe
- Budget range
- Next action date
- Owner
