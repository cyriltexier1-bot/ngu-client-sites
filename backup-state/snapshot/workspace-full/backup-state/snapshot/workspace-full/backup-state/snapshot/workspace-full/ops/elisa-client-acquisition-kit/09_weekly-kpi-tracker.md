# Weekly KPI Tracker

Track each week:

1. New leads received
2. Median first-response time
3. Contact rate (% reached)
4. Discovery calls booked
5. Booking rate (% of contacted)
6. Qualified leads created
7. Offers/listings progressed
8. Close rate
9. Reactivation attempts
10. Reactivation conversion rate

## Weekly review prompts
- What improved this week?
- Biggest bottleneck?
- Which source gave best quality leads?
- One process change for next week?

## Targets (starter)
- Response time: <15 min (business hours)
- Contact rate: >65%
- Booking rate: >35% of contacted
- Reactivation conversion: >8%
