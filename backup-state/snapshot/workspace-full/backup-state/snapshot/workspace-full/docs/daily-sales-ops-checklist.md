# Daily Sales Ops Checklist (Lead Hygiene & Follow-Up)

Purpose: keep pipeline clean, responsive, and conversion-focused.

## Daily (Mon–Fri, 30–45 min total)

### 1) Inbox & New Lead Triage (10 min)
- [ ] Review all new leads from last 24h (forms, calls, DMs, referrals).
- [ ] Assign owner for each lead.
- [ ] Set **Lead Source**, **Status**, and **Next Action Date**.
- [ ] Tag priority: **Hot / Warm / Cold**.

### 2) Data Hygiene Sweep (10 min)
- [ ] Merge duplicate contacts.
- [ ] Fix missing phone/email for active leads.
- [ ] Standardize notes: latest outcome + next step in one line.
- [ ] Archive/close dead leads with reason code (no response, lost, invalid, etc.).

### 3) Follow-Up Execution (15–20 min)
- [ ] Complete all follow-ups due today.
- [ ] For overdue leads: contact top 10 by deal potential first.
- [ ] Log every touchpoint (call, text, email) with timestamp + result.
- [ ] For no-response leads, schedule next touch using cadence (e.g., D+2, D+5, D+10).

### 4) End-of-Day Control (5 min)
- [ ] Ensure **0 unowned new leads**.
- [ ] Ensure **0 overdue hot leads**.
- [ ] Confirm tomorrow’s follow-up queue is populated.

---

## Weekly (60–90 min, e.g., Monday AM)

### 1) Pipeline Health Review
- [ ] Count leads by stage and compare vs last week.
- [ ] Identify stale leads (no activity in 7+ days) and assign recovery action.
- [ ] Re-score warm/cold leads based on latest engagement.

### 2) SLA & Speed-to-Lead Check
- [ ] Measure median first-response time.
- [ ] List leads waiting >24h for response and resolve backlog.
- [ ] Check owner workload balance; reassign if needed.

### 3) Follow-Up Quality Audit
- [ ] Spot-check 10 records for note quality and next-step clarity.
- [ ] Verify each active lead has a dated next action.
- [ ] Confirm automation sequences are active and error-free.

### 4) Source & Conversion Snapshot
- [ ] Review lead sources by volume and conversion trend.
- [ ] Flag low-quality sources for tightening or pause.
- [ ] Share top 3 improvements for the coming week.

---

## Pre-Listing Readiness Check (before going live)

### CRM & Lead Routing
- [ ] Listing record created with correct address, price, and timeline.
- [ ] Lead routing rules set (primary + backup owner).
- [ ] Response templates loaded (inquiry reply, viewing scheduling, FAQ).

### Follow-Up Infrastructure
- [ ] Cadence sequence prepared for first 14 days.
- [ ] Viewing request workflow tested end-to-end.
- [ ] Notification alerts enabled for new inquiries (SMS/email/app).

### Tracking & Reporting
- [ ] UTM/source tracking confirmed for all campaign links.
- [ ] Dashboard fields validated (source, stage, last contact, next action).
- [ ] Baseline KPI sheet ready: inquiries, response time, booked viewings, offers.

### Go/No-Go Gate
- [ ] Test lead submitted and received in CRM within SLA.
- [ ] First response sent correctly from assigned owner.
- [ ] Next action auto-created after first contact.
- [ ] If any check fails, fix before launch.

---

## Operating Rules (non-negotiables)
- Every active lead must have: **Owner + Status + Last Touch + Next Action Date**.
- If it isn’t logged in CRM, it didn’t happen.
- Hot leads are never left overdue overnight.
