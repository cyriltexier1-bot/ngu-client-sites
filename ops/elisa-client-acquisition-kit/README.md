# Elisa Client Acquisition Kit

Built to create immediate operational output (not just chat).

## Files

1. `01_email-to-elisa.txt`
   - Ready-to-send diagnostic email (yes/no + open questions)
2. `02_scoring-matrix.md`
   - How to score replies and detect bottlenecks quickly
3. `03_tailored-options-by-gap.md`
   - Recommendation menu mapped to weak areas
4. `04_followup-reply-templates.md`
   - Ready replies for common answer patterns
5. `05_daily-weekly-operating-system.md`
   - Practical daily/weekly cadence to improve lead conversion

## How to use

1. Send `01_email-to-elisa.txt`
2. Paste Elisa's answers
3. Score with `02_scoring-matrix.md`
4. Use `03_tailored-options-by-gap.md` + `04_followup-reply-templates.md`
5. Run execution with `05_daily-weekly-operating-system.md`
